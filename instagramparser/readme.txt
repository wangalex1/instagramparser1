Для запуска парсера необходимо:
1) В файле runner.py указать пользователя или список пользователей, которых нужно спарсить.
2) В файле spiders/instagram.py нужно в настройках указать логин и зашифрованный пароль от Instagram.

Парсер собирает всех подписчиков и все подписки пользователей из указанного списка.
Пользователи сохраняются в базу без дублирования. Один пользователь - один документ.
Подписки пользователя указываются в документе пользователя в списке.
В качестве _id используется id пользователя в Instagram.

Пример:
'_id': 846546548,
'name': 'cool_user',
'full_name': 'Ivanov Ivan Ivanovich',
'photo_url': 'https://scontent-arn2-1.cdninstagram.com/v/t5....',
'following': ['some_user', 'other_user', 'third_user']

В файле data_reader.py - программа для работы с записанной базой пользователей.
Функции: поиск пользователя, вывод подписчиков, вывод подписок.
